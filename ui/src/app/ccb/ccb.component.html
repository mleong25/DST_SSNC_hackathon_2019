Troubleshooting the bot Summer, 2019

About the bot

A part of our summer internship is an intern hackathon. For this year we decided to use a bot to integrate with MatterMost (the chat application we use daily). The purpose of this project was to increase automation and efficiency to improve the day to day lives of the employees that work with us. We wanted our bot to bridge the gap between MatterMost and Outlook, to be able to set up meetings through chats easily and quickly. When working on this project we came up with so many inspired ideas and we hope that we can continue improving this project even after the summer is over.
