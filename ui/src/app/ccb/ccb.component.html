Troubleshooting the bot Summer, 2019

About the bot

The Company Communication Bot is designed for SSNC use however can be easily customized for personal or general purpose. At its core, it runs a Java executable package that listens for a "ccb" command and attempts an API call to a target endpoint if it is valid. The bot was inspired by an ongoing joke between the interns on the PLEX Team about creating bots of each member of the team to auto-respond. The idea has since been flushed out and the bot is capable of much, much more!

Common Commands

All commands start with "ccb" followed by an action and the parameters expected by the action. A simple way to test if the bot is working on your machine is by typing "ccb test" into the message body to the bot. You will notice that the bot will respond immediately if the command was successful or nothing at all if it fails. The bot will also mention that a command is unknown if the user inputs an unknown or invalid command.
